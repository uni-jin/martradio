# 마트방송 관리자 사이트 기획/구현 문서

## 1) 목표
- 사용자 화면과 톤을 맞춘 심플/클린 관리자 UI 제공
- `admin / 123qwe`로 로그인 가능한 관리자 전용 영역 구성
- 회원/추천인/콘텐츠/결제/구독 관련 운영 기능 제공

## 2) 로그인/접근 정책
- 관리자 로그인 경로: `/admin/login`
- 계정: `admin`
- 비밀번호: `123qwe`
- 관리자 세션은 브라우저 `localStorage`에 저장
- `/admin/*` 경로는 관리자 세션 없으면 `/admin/login`으로 리다이렉트

## 3) 공통 레이아웃
- 상단 고정 바
  - 좌측: `마트방송 관리자` (클릭 시 `/admin`)
  - 우측: 로그인 관리자 계정 + 로그아웃
- 본문
  - depth1 메뉴(상단 뱃지형)
  - depth2 메뉴(좌측 사이드바)
  - 우측 본문 컨텐츠

## 4) 메뉴 구조

### 4-1. 회원관리
- 회원관리: `/admin/users`
  - 회원 목록 조회
  - 상세 페이지 이동
- 추천인관리: `/admin/referrers`
  - 추천인 추가/수정/삭제
  - 추천인별 가입자 수/결제 합계 기초 통계

### 4-2. 콘텐츠 관리
- 템플릿 관리: `/admin/templates`
  - 방송 템플릿 추가/수정/삭제
  - 유료 전용 여부 플래그
- 음성 템플릿 관리: `/admin/voices`
  - 사용자 화면에서 노출할 음성 활성/비활성 관리(기초 토글)

### 4-3. 결제/구독 관리
- 결제 상세 내역: `/admin/payments`
  - 결제 발생 일시/회원/금액/상품 조회
- 구독 상품 관리: `/admin/products`
  - 글자수 제한, 월 요금, 템플릿 사용 가능 여부 관리
- 추천인 결제 통계: `/admin/referrer-payments`
  - 추천인 기준 결제 합계/건수 확인

## 5) 데이터 저장 방식 (현재 단계)
- 별도 백엔드 전 단계로 `localStorage` 기반 운영 데이터 관리
- 향후 API/DB 전환 시 키 구조를 API 모델로 매핑 예정
